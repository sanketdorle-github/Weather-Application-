Project Description:

Welcome to weather application, a demonstration project designed to showcase my skills in web development and API integration. This app is created solely for the purpose of showcasing my abilities and is not intended for commercial use.

About Weather Application:

Weather Application is a simple yet powerful weather application that retrieves real-time weather data from reliable sources using APIs. It provides users with essential weather information for various cities worldwide. This includes current temperature, humidity, wind speed, and direction, as well as sunrise and sunset times. Additionally, users can easily switch between different cities to view their respective weather conditions.

Purpose and Goals:

The primary goal of Weather Application is to demonstrate my proficiency in front-end development, particularly in HTML, CSS, and JavaScript. It also showcases my ability to integrate third-party APIs seamlessly into web applications. Through this project, I aim to highlight my skills in creating user-friendly interfaces and handling asynchronous data fetching.

Disclaimer:

The weather data displayed within the app is sourced from external APIs and may not always be 100% accurate.
